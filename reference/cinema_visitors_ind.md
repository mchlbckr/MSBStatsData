# Cinema visitors over 100 days (individual-level)

Expanded version of
[cinema_visitors](https://mchlbckr.github.io/MSBStatsData/reference/cinema_visitors.md)
with one row per observed day.

## Usage

``` r
cinema_visitors_ind
```

## Format

A tibble with 100 rows and 1 variable:

- viewers:

  Number of visitors counted on the day.
