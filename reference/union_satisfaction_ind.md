# Satisfaction with a union agreement (individual-level)

The dataset contains the same information as
[union_satisfaction](https://mchlbckr.github.io/MSBStatsData/reference/union_satisfaction.md),
expanded to individual-level observations with one row per surveyed
person.

## Usage

``` r
union_satisfaction_ind
```

## Format

A tibble with 1,000 rows and 2 variables:

- union:

  Union identifier.

- satisfaction:

  Satisfaction category for the agreement.
