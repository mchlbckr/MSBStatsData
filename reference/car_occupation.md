# Car brand by occupation (aggregated)

The dataset contains counts of preferred car brands by occupation.

## Usage

``` r
car_occupation
```

## Format

A tibble with 4 rows and 6 variables:

- occupation:

  Occupation category.

- audi:

  Count of Audi drivers.

- bmw:

  Count of BMW drivers.

- opel:

  Count of Opel drivers.

- vw:

  Count of VW drivers.

- total:

  Row total.
