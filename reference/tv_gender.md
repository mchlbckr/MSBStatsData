# TV viewing time by gender (aggregated)

The dataset contains grouped TV viewing counts by gender.

## Usage

``` r
tv_gender
```

## Format

A tibble with 3 rows and 4 variables:

- tv_time_group:

  Grouped weekly TV viewing time and total row.

- men:

  Count of men.

- women:

  Count of women.

- total:

  Row total.
