# Decathlon summary for three disciplines

Mean and variance for 100m, long jump, and shot put from a large
decathlon sample.

## Usage

``` r
decathlon_3disc_summary
```

## Format

A tibble with 3 rows and 3 variables:

- discipline:

  Discipline name.

- mean:

  Arithmetic mean of results.

- variance:

  Variance of results.
