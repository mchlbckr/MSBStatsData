# Shipping error data

Dataset recording errors in the shipment and the type of delivery
requested by the customer in the process.

## Usage

``` r
shipping_errors
```

## Format

A tibble with 576 rows and 2 variables:

- error \[fct\]:

  Indicator whether or not an error appeared during shipment.

- shipping \[fct\]:

  Shipment method requested by the customer.
