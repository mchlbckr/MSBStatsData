# Grouped website dwell times (individual-level)

Expanded version of
[website_dwell](https://mchlbckr.github.io/MSBStatsData/reference/website_dwell.md)
with one row per sampled person.

## Usage

``` r
website_dwell_ind
```

## Format

A tibble with 100 rows and 1 variable:

- dwell_time_interval_min:

  Interval of dwell time in minutes for the person.
